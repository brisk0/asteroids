Linux
=====

Dependencies
------------
* SDL2
* Clang (for build)
* Gnu Make (for build)

Binary
------
	asteroids
Compiled for 64-bit Arch Linux

Compile
-------
	$ make release
	$ ./asteroids

Windows
=======

Dependencies
------------
* SDL2 (for build)
* MinGW (for build)
* Gnu Make (for build)

Binary
------
	windows/asteroids.exe
Compiled for 32-bit Windows

Compile
-------
	> make win32

Controls
========
* Left/Right: Turn ship
* Space: Fire bullet
* R: Restart at end of game
