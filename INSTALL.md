Linux
=====

Dependencies
------------
* SDL2
* Clang (for build)
* Gnu Make (for build)

Binary
------
	kuiper
Compiled for 64-bit Arch Linux

Compile
-------
	$ make release

Windows
=======

Dependencies
------------
* SDL2 (for build)
* MinGW (for build)
* Gnu Make (for build)

Binary
------
	windows/kuiper.exe
Compiled for 32-bit Windows

Compile
-------
	> make win32

Controls
========
* Left/Right: Turn ship
* Space: Fire bullet
* R: Restart at end of game
